Here’s a simple Python approach you could put in a notebook or script. It loads the CSV, removes wetlands, calculates a score, and returns the best 4 segments.

Python Example
import pandas as pd

# Load your CSV
df = pd.read_csv("segments.csv")

# 1. Remove wetlands (exclusion rule)
df = df[df["WETLANDS_NEARBY"] == 0]

# 2. Create a score (lower = better)
df["score"] = (
    df["CRASH_RATE"] +
    df["INCIDENT_RATE"] +
    df["WEATHER_RISK_AREA"] +
    df["PEAK_TIME_BUCKET"] +
    df["INTERCHANGE_RATES"] +
    df["LOW_TRUCK_USAGE"] -
    df["HIGH_EV_VEHICLE_USAGE"]
)

# 3. Sort by score
df_sorted = df.sort_values("score")

# 4. Select best 4
best_sites = df_sorted.head(4)

print(best_sites[["SEGMENT_ID", "score"]])
