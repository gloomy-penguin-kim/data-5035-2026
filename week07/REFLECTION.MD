# Reflection 

## 1. Which programming paradigm you used where

- Declarative paradigm: SQL in Snowflake for joins, filtering, window functions, and geospatial logic
- Imperative paradigm: Python in Snowflake notebook for filtering, scoring, sorting, and CSV export
- AI paradigm: AI used for debugging, refactoring, and improving code structure

## 2. Why you chose that paradigm

- SQL was best for set-based data processing and geospatial calculations
- Python was used for step-by-step scoring logic and to satisfy the imperative requirement
- AI was useful for troubleshooting, refactoring, and improving clarity
